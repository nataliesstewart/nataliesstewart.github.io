I got back to MIT yesterday. I want to talk about an easy but seemingly
important structural result. Today, I want to show that the Atiyah-Hirzebruch
spectral sequence for $$\CP^\infty$$ with coefficients in any $$E_n$$-local
homotopy commutative ring spectrum collapses *at a finite page*. Ultimately,
this degeneration takes place for the same reason as the degeneration of the
Adams-Novikov spectral sequence for any $$E_n$$-local homotopy commutative ring
spectrum.

Hopkins and Ravenel show that the $$E_n$$-local sphere $$L_n \S$$ is in the
thick subcategory generated by the spectra $$E_n^{\wedge s}$$ for $$s\geq 1$$.
Every $$E_n$$-local homotopy commutative ring spectrum admits a ring map from
$$L_n \S$$, which induces a map of Atiyah-Hirzebruch spectral sequences. So all
we need to show is that if $$F$$ is in the thick subcategory generated by $$E$$
and the $$E$$-based Atiyah-Hirzebruch spectral sequence collapses, then so does
the $$F$$-based one.  In other words, it therefore suffices to show that the
property of having the Atiyah-Hirzebruch spectral sequence vanishing at a finite
page is a "generic" property, i.e., is closed under the operations defining a
thick subcategory.

There is a cosimplicial spectrum $$Y_\bullet$$ defined by the bar construction
on $$S^1$$. This defines a filtered spectrum $$X_E:\mathrm{N}(\Z_{\geq 0})^{op}
\to \Sp$$ given by $$X_E(s) = \Tot^s E^{Y_\bullet} \simeq E^{\CP^s}$$, which
defines an associated spectral sequence (the Bousfield-Kan spectral sequence)
that is isomorphic from the second page onwards to the $$E$$-based
Atiyah-Hirzebruch spectral sequence. If $$F$$ is in the thick subcategory
generated by $$E$$, then $$X_F$$ is in the thick subcategory of
$$\Fun(\mathrm{N}(\Z_{\geq 0})^{op}, \Sp)$$ generated by $$X_E$$. 
  
By Proposition 3 of [this
lecture](http://www.math.harvard.edu/~lurie/252xnotes/Lecture30.pdf) (which is
in fact an if-and-only-if statement), it suffices to show that the tower $$X_E$$
is pro-constant. In light of the above discussion, it therefore suffices to
prove the following statement: let $$\cC$$ be a complete stable
$$\infty$$-category, and let $$\Fun^\fin(\mathrm{N}(\Z_{\geq 0})^{op}, \cC)$$
denote the category of functors $$X:\mathrm{N}(\Z_{\geq 0})^{op}\to \cC$$ for
which the sequence $$\{X(n)\}$$ is pro-constant. Then
$$\Fun^\fin(\mathrm{N}(\Z_{\geq 0})^{op}, \cC)$$ is a thick subcategory of
$$\Fun(\mathrm{N}(\Z_{\geq 0})^{op}, \cC)$$.

This is proved (but not stated in the same level of generality) in [this
paper](https://arxiv.org/abs/math/9907198) by Hopkins-Palmieri-Smith, but we
recall the proof here anyway. I'll give another proof (which maybe works modulo
silly mistakes) for $$\cC = \Sp$$ below. We can reduce to the case of pro-zero
objects $$\{X(n)\}$$, since if $$\{X(n)\}$$ is pro-constant then the cofiber of
the map $$\{\lim X(n)\} \to \{X(n)\}$$ is pro-zero. We need to show that if
$$\{X(n)\} \to \{Y(n)\} \to \{Z(n)\}$$ is any cofiber sequence of objects of
$$\Fun(\mathrm{N}(\Z_{\geq 0})^{op}, \cC)$$ with $$\{X(n)\}$$ and $$\{Y(n)\}$$
in $$\Fun^\fin(\mathrm{N}(\Z_{\geq 0})^{op}, \cC)$$, then there is some $$m>0$$
such that the map $$\Map(C, Z(n))_\ast \to \Map(C,Z(n-m))_\ast$$ is zero for
every $$C\in \cC$$. Since $$\{X(n)\}$$ and $$\{Y(n)\}$$ are both pro-zero, there
are some $$m',m''>0$$ such that the maps $$\Map(C, X(n))_\ast \to
\Map(C,X(n-m'))_\ast$$ and $$\Map(C, Y(n))_\ast \to \Map(C, Y(n-m''))_\ast$$ are
zero for every $$C\in \cC$$. There is a commutative diagram

  <img style="display: block; margin: auto;" height="75%" width="75%"
  alt="photo" src="/assets/ahss.png">

Let $$\Sigma^d C \to Z(n+m'+m'')$$ be any map; then, by commutativity of the
above diagram, the map

$$\Sigma^d C\to Z(n+m'+m'') \to Z(n+m'') \to \Sigma X(n+m'')$$

is zero, since any map $$\Sigma^d C \to \Sigma X(n+m'+m'') \to \Sigma X(n+m'')$$
vanishes. Therefore, it factors through $$Y(n+m'')$$. Again using commutativity,
the composite

$$\Sigma^d C\to Z(n+m'+m'') \to Y(n+m'') \to Z(n+m'') \to Z(n)$$

must be null, since any map $$\Sigma^d C \to Y(n+m'') \to Y(n)$$ is null. It
follows that by setting $$m=m'+m''$$, the map $$\Map(C, Z(n))_\ast \to
\Map(C,Z(n-m))_\ast$$ is zero for every $$C\in \cC$$, as desired.

I'll end by providing another proof of the claim that if
$$\Fun^\fin(\mathrm{N}(\Z_{\geq 0})^{op}, \cC)$$ denotes the category of
functors $$X:\mathrm{N}(\Z_{\geq 0})^{op}\to \cC$$ for which the sequence
$$\{X(n)\}$$ is pro-constant, then $$\Fun^\fin(\mathrm{N}(\Z_{\geq 0})^{op},
\cC)$$ is a thick subcategory of $$\Fun(\mathrm{N}(\Z_{\geq 0})^{op}, \cC)$$.
We'll begin by recalling the construction of the associated spectral sequence.
For $$a\geq b$$, define $$X(a,b)$$ to be the fiber of the map $$X(a) \to X(b)$$.
Then, $$E_r^{p,q}(X)$$ is the image of the map $$\pi_{q-p} X(p+r-1,p-1) \to
\pi_{q-p} X(p,p-r)$$. There are fiber sequences

$$X(p,p-r) \to X(p,p-2r) \to X(p-r,p-2r)$$

and

$$X(p+r-1,p-1) \to X(p+r-1,p-r-1) \to X(p-1,p-r-1),$$

which give rise to a differential

  <img style="display: block; margin: auto;" height="75%" width="75%"
  alt="photo" src="/assets/ahss-2.png">

It is easy to show that $$\Fun^\fin(\mathrm{N}(\Z_{\geq 0})^{op}, \cC)$$ is closed
under suspensions and retracts, so we prove that it is closed under cofiber
sequences. Let $$X\xrightarrow{i} Y\xrightarrow{j} Z$$ be a cofiber sequence in
$$\Fun(\mathrm{N}(\Z_{\geq 0})^{op}, \cC)$$ with $$X$$ and $$Z$$ in
$$\Fun^\fin(\mathrm{N}(\Z_{\geq 0})^{op}, \cC)$$. Then, for $$r'\geq r$$, we
have the following commutative diagram:

  <img style="display: block; margin: auto;" height="75%" width="75%"
  alt="photo" src="/assets/ahss-3.png">

Since $$X(i) = 0$$ for $$i<0$$, we can identify $$X(p,p-r)$$ with $$X(p)$$ for
$$r>p$$. We will be assuming that $$r$$ is very large, so we will simply write
$$X(p)$$ instead of $$X(p,p-r)$$. Since $$X$$ is pro-constant, there is some
$$N\geq 0$$ such that the maps $$\pi_{q-p} X(p+r-1,p-1) \to \pi_{q-p} X(p)$$ and
$$\pi_{q-p} X(p+r'-1,p-1) \to \pi_{q-p} X(p+r-1,p-1) \to \pi_{q-p} X(p)$$ have
isomorphic images (namely, $$E_r^{p,q}(X) \cong E_{r'}^{p,q}(X)$$ for $$r'\geq
r\geq N$$). We would like to prove that the same is true for $$Y$$.  Suppose
that there is no $$M\geq 0$$ such that $$E_r^{p,q}(Y) \cong E_{r'}^{p,q}(Y)$$
for $$r'\geq r\geq M$$, so that there is $$\alpha\in\im(f)$$ which is not in
$$\im(fg)$$. In other words, there is no element in $$\pi_{q-p} Y(p+r'-1,p-1)$$
whose image under $$g$$ lives in $$f^{-1}(\alpha)$$. Let $$\beta\in
f^{-1}(\alpha)$$.

Either $$j(\alpha) \in \im(f'' g'')$$ or $$j(\alpha)\not\in \im(f'' g'')$$.
Since $$Z$$ is also pro-constant, by choosing $$r'$$ and $$r$$ large enough,
the second possibility can be eliminated, so $$j(\alpha) \in \im(f'' g'')$$.  It
follows that there is $$\gamma\in \pi_{q-p} Z(p+r'-1,p-1)$$ such that
$$g''(\gamma) = j(\beta)$$.  By exactness, if there is no $$\beta\in
f^{-1}(\alpha)$$ which is not in $$\im(g)$$, we need $$\delta(\gamma) \neq 0$$.
But

$$0 = \delta j(\beta) = \delta g''(\gamma) = g'\delta(\gamma),$$

so $$\delta(\gamma)\in \ker(g')$$. The fiber of the map $$X(p+r'-1,p-1) \to
X(p+r-1,p-1)$$ is $$X(p+r'-1,p+r-1)$$, so $$\delta(\gamma)\in \pi_{q-p}
X(p+r'-1,p+r-1)$$. This is zero for $$r'$$ and $$r$$ large enough, since $$X$$
is pro-constant. It follows that $$\delta(\gamma) = 0$$ for $$r'$$ and $$r$$
large enough, so there is some $$\beta\in f^{-1}(\alpha)$$ which is in
$$\im(g)$$, as desired.
