I suggested in [this post](/blog/algebraic-topology/2020/04/24/may.html) that it
may be possible to prove May nilpotence by a Devinatz-Hopkins-Smith style
nilpotence argument, by going up the symmetric power filtration of $$\H\Z$$. The
suggestion in that post doesn't exactly work as written, but
[Jeremy](http://web.mit.edu/~jhahn01/www/) observed a fix in a special case that
might generalize. I'd like to explain this observation in this post.

First, setup. Let $$R$$ be an $$\Eoo$$-ring, and let $$\H\Z \otimes R = 0$$. We
want to show that $$R = 0$$. Let's localize at $$2$$. Note that $$R\otimes \H\QQ
= 0$$, so we can assume that $$2^n = 0$$ in $$\pi_0 R$$ for some $$n$$. Then, we
just need to show that $$2^{n-1} = 0$$.

Now, the symmetric power filtration $$\Sym^{2^n}(\S) \to \H\Z$$ is filtered, so
the unit $$\S \to R\to R\otimes \H\Z$$ factors through some $$\Sym^{2^n}(\S)
\otimes R$$. Jeremy's argument assumes that the unit factors through
$$\Sym^2(\S)\otimes R$$. Since $$R\otimes \H\Z$$ is zero, the unit is null, and
so the map $$\S\to R\to \Sym^2(\S)\otimes R$$ is null. *If* $$\Sym^2(\S)\otimes
R$$ was a ring, then this would immediately imply that $$\Sym^2(\S)\otimes R$$
is zero; but it's not (and this was the problem in the previous post).

*However*, one can still argue that if the map $$\S\to R\to \Sym^2(\S)\otimes
R$$ is null (and $$2^n = 0\in \pi_0(R)$$), then $$2^{n-1} = 0\in \pi_0(R)$$. To
see this, note that there is a cofiber sequence

$$\RP^\infty \xrightarrow{\tr} \S \to \Sym^2(\S),$$

and so the fact that the unit in $$R\otimes \Sym^2(\S)$$ is null implies that
the unit map $$\S\to R$$ lifts along the transfer $$\tr:\RP^\infty \otimes R
\to R$$; in other words, there is a map $$f:\S\to \RP^\infty \otimes R$$ such
that the composite

$$\S\xrightarrow{f} \RP^\infty \otimes R \xrightarrow{\tr} R$$

is the unit.

Now, we know that $$2^n = 0\in \pi_0(R)$$. In general, if $$N: \pi_0(R) \to
\pi_0(R^{\RP^\infty_+})$$ is the norm, then there is a relation $$N(x+y) =
N(x)+N(y) + \tr(xy)$$. At an odd prime, we get

$$N(x+y) = N(x) + N(y) + \Tr\left(\sum_{i\leq p-i} \frac{1}{p} \binom{p}{i} x^i
y^{p-i}\right);$$

this is basically the binomial theorem. In any case, we see that

$$N(a) = a + \frac{a^p - a}{p} \Tr(1)$$

for any integer $$a$$, so $$N(2^n) \equiv 2^{n-1}\tr(1) \pmod{2^n}$$, and thus
$$\tr(2^{n-1}) = 0$$ in $$R$$. Since the unit of $$R$$ factors through the
transfer, this forces $$2^{n-1} = 0\in \pi_0(R)$$, as desired.

In a future post, we will discuss what happens if the unit factors through
$$R\otimes \Sym^4(\S)$$ instead.
